## Project Overview
This repository contains a data analysis project focused on exploring historical sales data 
from a supermarket chain to uncover valuable business insights and optimize sales strategies. 
The project culminates in an interactive dashboard that visualizes key performance indicators (KPIs) and trends.
## Problem statements
The supermarket aims to understand its sales performance across different branches and product lines, identify customer purchasing patterns, and leverage these insights to make data-driven decisions that increase revenueand operational efficiency.
## Key Features
**Data Cleaning and Preparation:** Handled missing values, removed duplicates, and transformed raw data into a structured format suitable for analysis.

**Exploratory Data Analysis (EDA):** Identified key sales trends, peak customer hours, high-value product lines, and purchasing patterns across different demographics.

**KPI Visualization:** Developed an interactive dashboard to track total sales, gross profit, cost of goods sold (COGS), and average customer ratings.

**Actionable Insights:** Provided data-driven recommendations to management to optimize product placement and promotional activities.

## Tools and Technologies

**Databases:** SQL (for data extraction and preliminary analysis)

**Programming:** Python (Pandas, NumPy for data manipulation and analysis)

**Visualization:** Power BI (interactive dashboard creation)

**Version Control:** Git & GitHub
